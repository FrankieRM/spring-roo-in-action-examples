spring-roo-in-action-examples by krimple

Spring Roo in Action Example code - Manning Publications Ken Rimple, Srini Penchikala, Gordon Dickens, authors

This project represents all example code for the Spring Roo in Action book by Manning Publications, estimated publication date December 2011. The samples are currently being loaded to this repository, and we will accept suggestions for revisions once we are finished publishing our initial version.

Dependencies

Spring Roo 1.2.M1, preferrably 1.2.BUILD-SNAPSHOT, or higher.

Install

Each of the projects in this repo has its own build file, so just cd to the project and execute `mvn package` or to run a web application `mvn jetty:run`

License

All code contributed as sample code for Spring Roo in Action is contributed to the community in support of the book. You may cut/paste and use the samples at will; we only ask you credit the book with attribution if you use these samples on a web site.
